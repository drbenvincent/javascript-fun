Viral spread simulator

Made by @inferenceLab

I was really impressed with the Corona virus simulations which appeared in The Washington Post recently. So much so that it inspired me to learn JavaScript. So this is my first attempt at learning html, css, and JavaScript.

This implements a Susceptible-Infected-Recovered (SIR) model in a small finite population of agents moving about at random. All agents (apart from patient zero) starts off susceptible. An agent will get infected if they come into contact with an infected agent.

N = number of balls

For fun and educational purposes only. Do not make any life decisions based upon this simulation. Code is available on my GitHub.
